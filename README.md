This is the 2025-2026 coursework from Smart Internet Technologies module (Computer Science)

# Github Experiemnt Summary
The intention of this experiment was to:
 * To trancode two videos into three seperate bitrates(1.5Mbps,2.0Mbps & 4.0Mbps)
 * To create a DASH manifest, which has a .mpd file output
 * To establish an IPERF TCP connection betweent he VM Server and VM Client with assigned highest priority in the connection
 * A webserver for the VM Server and VM Client to communicate solely through the internet
 * Emulate Network atefacts such as TBF (Token Bucket Filter), HTB (Hierarchal Token Bucket) and Traffic Policing
 * Determine MOS(Mean Opinion Score) for each Network Artefact
 *  Use MOS to see how QoE (Quality Of Experience) is improved

In this Github directory there will the 6 transcoded videos in their respective bitratesl; one set of three would be from Videezy website, whilst the second set of three would come from Pexels website.
